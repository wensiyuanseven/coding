# Vant Weapp 示例

## 目录

### base

示范了如何用 Vant Weapp 搭建几个简单的电商小程序页面，包含如下功能：

- 基于 Vant Weapp 搭建小程序应用
- 样式覆盖方案

> 技术栈：Vant Weapp

## 开发命令

```bash
# 安装依赖
# 注意：请切换到对应的子目录下安装
cd base
npm install

# 打开微信开发者工具，导入示例工程，点击『工具』 -> 『构建 npm』，并勾选『使用 npm 模块』选项即可
```
